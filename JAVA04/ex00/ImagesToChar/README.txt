
1. to compile the program, run "make" command.

2. to run the program, use "java -cp target -white=${char} -black=${char} -path={pathToBmpImage}".

3. put the character representing the white in the value of "-white", put the character representing the black in the value of "-black".
    In the "-path", put the absolute path of the image in your computer.

4. The program may not work properly if the instructions were not followed.
